# Provisional Patent Filing Checklist

## Required Documents
- [ ] Cover sheet (USPTO Form SB/16)
- [ ] Specification (detailed description)
- [ ] Claims (at least 1 independent claim)
- [ ] Abstract (150 words or less)
- [ ] Drawings (if applicable)
- [ ] Filing fee ($300 for micro entity, $150 if qualify)

## Optional but Recommended
- [ ] Inventor declaration
- [ ] Prior art search results
- [ ] Source code samples
- [ ] Technical diagrams
- [ ] Evidence of working implementation

## Filing Options
- [ ] USPTO.gov online filing (fastest)
- [ ] Patent attorney (recommended)
- [ ] Patent agent
- [ ] Self-filing

## After Filing
- [ ] Record filing date and application number
- [ ] Mark all materials "Patent Pending"
- [ ] File full utility patent within 12 months
- [ ] Continue developing/documenting invention
