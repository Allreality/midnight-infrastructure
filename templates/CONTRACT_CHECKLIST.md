# Contract Preparation Checklist

## Before Sending Contract

### For Free Assessment
- [ ] Fill in client company name
- [ ] Fill in client address
- [ ] Fill in effective date
- [ ] Review scope matches what you discussed
- [ ] Attach NDA if client requested
- [ ] Save as PDF with filename: TRG_Assessment_[ClientName]_[Date].pdf

### For Paid Services
- [ ] Fill in client company name and address
- [ ] Fill in effective date
- [ ] Select correct service tier (Professional/Enterprise/Custom)
- [ ] Select payment option
- [ ] Fill in project timeline
- [ ] Complete Exhibit A (Statement of Work)
- [ ] List all deliverables
- [ ] Define milestones
- [ ] Save as PDF: TRG_Services_[ClientName]_[Date].pdf

## Sending Contract

### Email Template
```
Subject: Agreement - Security [Assessment/Implementation]

Hi [Name],

Attached is the agreement for our [assessment/implementation project].

**Key Points:**
- [Service tier and price]
- [Timeline]
- [Key deliverables]

**Next Steps:**
1. Review agreement
2. Sign both copies
3. Return one signed copy (scan/email is fine)
4. [If paid: Issue PO or send first payment]
5. We'll schedule kickoff within 48 hours

Questions? Happy to discuss any section.

Looking forward to working together!

Best,
Akil
```

## After Contract Signed

- [ ] Countersign your copy
- [ ] Send fully executed copy to client
- [ ] Create project folder: /assessments/[client-name]
- [ ] Add to Assessment Tracker
- [ ] Schedule kickoff call
- [ ] Send calendar invites
- [ ] If paid: Confirm payment received
- [ ] Begin work!
