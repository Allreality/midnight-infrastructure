# Midnight Blockchain Overview and Architecture

Research Findings: Midnight Blockchain Architecture

1. Overview
Midnight is a privacy-preserving blockchain platform developed by Input Output Global (IOG) as a partner chain to Cardano, designed to enable confidential computing and data protection while maintaining regulatory compliance.

2. Core Architectural Characteristics

Layer Classification:
- Technically a Layer 1 blockchain
- Operates as a partner chain to Cardano's main network
- Utilizes zero-knowledge proof (ZKP) infrastructure

Key Technical Specifications:
- Consensus Mechanism: Proof-of-Stake (PoS)
- Cryptographic Foundation: Zero-knowledge proofs
- Programming Language: TypeScript-based KERI (Key Event Receipt Infrastructure)
- Transaction Privacy: Selective data disclosure
- Smart Contract Execution: Confidential computation

3. Unique Privacy Architecture

Zero-Knowledge Proof Implementation:
- Uses advanced ZKP protocols allowing:
  - Transaction verification without revealing transaction details
  - Cryptographic proof of data validity
  - Selective information disclosure
- Enables compliance with regulatory requirements
- Maintains transaction integrity without compromising confidentiality

4. Differentiation from Other Privacy Blockchains

Distinctive Features:
- Native Cardano integration
- Regulatory-friendly privacy model
- Developer-friendly TypeScript programming
- Enterprise-grade confidential computing
- Granular privacy controls
